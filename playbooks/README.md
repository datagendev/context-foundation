# Playbooks

Company documentation and context indexed for AI-powered personalization.

## Files

### company-docs-llama-index.md
Guide for setting up Llama Index to index your company documentation:
- Product documentation
- Sales playbooks
- Case studies
- Company values and story
- Pricing and packaging

Use case: Use Llama Index to create company context that can be referenced in AI-powered email generation and campaign creation.

## How It Works

1. **Index your documents** with Llama Index
2. **Reference in AI workflows** via `/workflows/create_company_info`
3. **Personalize outreach** using company context
4. **Maintain single source of truth** for company information

## Key Integration

- **Outreach campaigns**: Reference company docs for personalization
- **Lead context**: Add company story to prospect research
- **Playbook generation**: Auto-generate pitches from documented playbooks
