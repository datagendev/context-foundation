# TAM - Total Addressable Market

This folder contains all tools and strategies for **lead sourcing** - finding your Total Addressable Market.

## Folders

### linkedin/
- **crust-data.md** - LinkedIn data integration via crustData
- **search-queries.md** - Sales Navigator query patterns and strategies

### ocean-io/
B2B data sources from Ocean.io

### panda-match/
B2B data enrichment via PandaMatch

### ciro/
Alternative B2B intelligence source

### serp-firecrawl/
Web scraping and data extraction tools:
- **brightdata.md** - BrightData scraping network
- **apify.md** - Apify web scraping actors
- **google-maps-mcp.md** - Google Maps MCP integration
- **exa-parallel.md** - Exa parallel processing for web data

### sourcing-playbooks/
Strategic approaches to lead generation:
- **industry-specific.md** - Industry-based targeting strategies
- **intent-signals.md** - Intent-based lead discovery

## How to Use

1. **Start with your ICP** (see `/icp/personas.md`)
2. **Choose TAM sources** based on your data needs (LinkedIn, web scraping, etc.)
3. **Use playbooks** to structure your sourcing strategy
4. **Output leads** to your CRM (see `/crm/`)
5. **Track with dashboards** (see `/dashboards/`)
